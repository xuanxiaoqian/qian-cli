# 安装

使用npm全局安装:

~~~sh
npm i qian-cli -g
~~~
<br />

初始化一个`项目`:

~~~sh
qian-cli init my-project
~~~

<vua-image src='/cli-init.png' />
<br />

初始化远程git`模板`:

~~~sh
qian-cli init git-project <你的仓库地址> [指定分支]
~~~

<vua-image src='/cli-init-git.png' />
<br />

创建一个`模块`:

~~~sh
qian-cli create moduleName
~~~

<vua-image src='/cli-create.png'/>
