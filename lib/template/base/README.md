## 介绍

传统的`vue-cli`、`create-vue`创建出来的项目功能都比较基础，不能满足开发需求，qian-cli在前者的基础上进一步配置，达到创建无需配置即可开发。



qian-cli还提供了快捷生成文件，无需手动创建文件、文件夹。



## 特点

- 便捷 一行简单命令即可完成繁琐配置

- 可选 选你所选 想你所想 懂你所懂

- 灵活 创建项目后，还能继续使用脚手架创建一系列模块

- 最新 开发模板都会维护，与时俱进





## 安装

使用npm全局安装:

~~~sh
npm i qian-cli -g
~~~
<br />

初始化一个`项目`:

~~~sh
qian-cli init my-project
~~~
<br />

创建一个`模块`:

~~~sh
qian-cli create moduleName
~~~
